# Assinador Digital Web

Este é um projeto acadêmico de um Assinador Digital Web utilizando criptografia assimétrica (RSA) e funções de hash (SHA-256).

## 🛠️ Tecnologias Utilizadas
* **Backend:** Node.js, Express, Cors, Crypto (nativo).
* **Banco de Dados:** SQLite (arquivo local, sem necessidade de containers complexos).
* **Frontend:** HTML5, JavaScript (Vanilla com Fetch API).

## 🚀 Como Rodar o Projeto

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone o repositório e navegue até a pasta do `backend`:
   ```bash
   cd backend
   npm install
   npm start
